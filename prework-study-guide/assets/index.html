HTML

  • The head element cannot be seen by users
  • The body element is displayed on the webpage
  • h1 through h6 rank the importance of headings
  • li and ul are used to list things
  • The footer is the section at the bottom of the page

CSS

  • You must link css file to the head of the HTML to display your css
  • A margin indicates how much space around the outisde of an element
  • A padding indicates how much space around the content inside an element
  • Assigning colors can be done in different ways
  • width can be defined by pixels or percentages

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • A variable is a named container that allows us to store data in our code
  • Control flow is the order in which a computer executes code in a script.
  • A function must be called out in the code in order to be executed.
  • Javascript uses if statements, and for loops.
